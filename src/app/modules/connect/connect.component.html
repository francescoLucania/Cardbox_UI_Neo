Подключение и использования theme_provider - scss-конфигурация для системного написания стилей проекта

Содержание theme_provider

в epgu-lib/src/assets/styles theme-provider добавлена папка theme-provider - scss конфигурация из переменных и миксинов:

  • variables: переменные(цвета, отступы, брекпоинты) по Сardbox UI
  • mixins: миксины конфигурации, упрощают работу с брекпоинтами, позволяют автоматизиривать некоторые аспекты работы с версткой
  • global-styles: небольшой "начальный" набор стилей документа, подключается по желанию, в _theme-provider.scss поумолчанию не импортирован
  • reset_styles: сброс браузерных стилей, кнопки, формы, подключается по желанию, в _theme-provider.scss по умолчанию не импортирован
  • helpers: немного атомарных классов для работы с брекпоинтами и текстом
  • components: links, buttons, typography: миксины и атомарные классы по ключевым элементам дизайн системы
